Estructura de un Prompt

1. Instrucción

Especifica lo que quieres que haga el modelo de IA, debe ser clara y precisa.

2. Contexto

Proporciona el marco o la circunstancia en la que se debe realizar la tarea, esto ayuda a que la IA entienda mejor la solicitud.

3. Input

Información adicional o detalles específicos que se deben tener en cuenta al generar la respuesta.

4. Output

Indica cómo debería ser la respuesta, por ejemplo, el formato o el tipo de información esperada.
